## Prerequisites

- git
- docker

## Get Your ALIS ID Token

### Case1: One liner

`$ curl -sL https://github.com/alisista/alis-id-token/archive/master.tar.gz | tar xvz "*" && cd alis-id-token-master && make show`

### Case2: make

```
$ git clone https://github.com/alisista/alis-id-token
$ cd alis-id-token
$ make show
```
